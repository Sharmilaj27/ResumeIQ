ResumeIQ – AI Resume Analyzer and Job Matcher

Overview

ResumeIQ is a web application that analyzes resumes using AI and provides structured feedback to improve quality and alignment with job roles. It evaluates resumes based on ATS (Applicant Tracking System) criteria, identifies strengths and weaknesses, and optionally compares them against a job description.


Features

* Resume upload (PDF, DOCX, TXT)
* AI-based resume analysis
* ATS score and overall evaluation
* Keyword and skills assessment
* Job description matching
* Actionable improvement suggestions

 Tech Stack

* Frontend: React, TypeScript, Tailwind CSS
* Backend: Supabase (Edge Functions)
* AI Integration: LLM-based analysis
* Deployment: Vercel

 Usage

1. Upload a resume
2. Optionally provide a job description
3. Select a target role
4. Run the analysis to receive structured feedback and scores

 Privacy

User data is processed securely and is not stored permanently.
